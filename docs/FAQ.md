# 常见问题解答（FAQ）

## 1. 这个工具和真实答辩有什么区别？

**答：** 模拟答辩工具旨在提供一个接近真实答辩环境的练习场景，但二者存在本质区别：

- **真实性方面**：真实答辩由经验丰富的教授或行业专家担任评委，他们拥有实际的学术背景和领域洞察力，能够结合现场表现给出综合判断。AI 评委虽然能模拟提问和评分，但无法完全复现真人评委的临场应变、表情语调和情绪感知。
- **反馈深度**：真人评委能根据你的现场答辩表现（如语言表达、肢体语言、回答流畅度）给出即时反馈，而 AI 评委主要基于论文内容和回答文本进行评判。
- **社交压力**：真实答辩存在不可忽略的心理压力和社交氛围，这是模拟环境难以完全复制的。

**总结：** 本工具是高质量的自助练习手段，适合用于熟悉答辩流程、打磨回答思路、检验论文薄弱环节，但不能替代真实答辩的体验。

---

## 2. AI 评委的提问靠谱吗？

**答：** 整体上比较靠谱，但也存在局限。

- **靠谱之处**：AI 评委能基于你提供的论文摘要和内容，结合通用的学术评审框架（如研究意义、方法论合理性、创新点、论证逻辑等），生成结构合理、覆盖全面的提问。这些提问往往是真实答辩中评委关注的核心问题，具有较高的参考价值。
- **局限**：AI 评委无法像领域专家那样识别出非常细分的专业漏洞或前沿文献引用偏差。它也可能偶尔提出过于宽泛或偏离你研究方向的提问。

**建议：** 将 AI 提问视为"大概率会遇到的问题"而非"一定会被问到的精确问题"，重点训练回答思路而非死记硬背答案。

## 3. 如何让 AI 评委给出更精准的评价？

**答：** 以下方法可以显著提升评价质量：

1. **提供高质量的论文摘要**：摘要应结构清晰、逻辑完整，包含研究背景、问题陈述、研究方法、核心发现和结论。信息越充分，AI 的提问和评价越有针对性。
2. **明确研究方向与学科**：在设置中指定你的学科领域（如计算机科学、管理学、教育学等），让 AI 评委在对应框架下进行评审。
3. **提供具体章节内容**：如果条件允许，输入论文的关键章节（如研究方法、数据分析部分），AI 对这些具体内容的评价最为精准。
4. **迭代追问**：对 AI 评委的某条评价不满意时，可以进一步追问（例如"能否从方法论角度给出更详细的批评？"），促使 AI 深化分析。
5. **交叉使用不同平台**：不同 AI 模型在学术评审上的侧重点不同，综合多个模型的反馈可以获得更全面的评价。

## 4. 我的论文数据会不会被泄露？

**答：** 数据安全是本工具关注的核心问题之一。

- **本地处理优先**：本工具的评分逻辑和部分基础评审功能在本地运行，论文内容不会上传至外部服务器。
- **第三方 API**：当选择使用云端大模型平台（如 OpenAI、Claude、文心一言等）时，论文摘要等信息会通过 API 传输至对应模型进行处理。建议在使用前查阅相应平台的隐私政策。
- **建议措施**：
  - 避免上传包含个人敏感信息（如姓名、学号、联系方式）的论文版本。
  - 对关键数据进行适当脱敏处理。
  - 如对安全性有极高要求，可使用支持本地部署的大模型。

## 5. 需要付费吗？

**答：** 本工具本身是开源免费的，你可以自由下载和使用。

但是，部分功能可能涉及第三方大模型 API 调用，这些 API 服务由对应的模型平台（如 OpenAI、Anthropic、百度等）收费，费用由相应平台收取，与本工具无关。

- 如果使用完全本地运行的开源模型（如通过 Ollama 工具调用本地大模型），则无需任何额外费用。
- 如果使用云端模型，费用取决于你的 API 调用量和对应平台的定价策略。

## 7. 如果我和 AI 评委意见不一致怎么办？

**答：** 这是正常的，处理方式建议如下：

1. **理性分析**：先审视 AI 评委的意见是否有合理依据。即使是 AI 的"误判"，也可能反映出你论文中某些表述确实容易被误解，这本身就是一个改进信号。
2. **保留自主判断**：你是自己论文最了解的人。如果确信 AI 的评价存在偏差，可以思考如何通过修改论文表述来消除这种误解，而不是盲目照单全收。
3. **寻求交叉验证**：用不同 AI 平台对同一段内容进行评审，如果多个平台给出相似意见，则说明确实存在问题。如果只有一家持有异议，则可能是模型偏见。
4. **回归导师建议**：最终的修改决策应以导师的意见和学术共同体的普遍标准为依归。AI 评委是辅助工具，不应成为论文修改的唯一依据。

## 8. 这个工具能替代导师指导吗？

**答：** **绝对不能替代。** 本工具的定位是"辅助练习与自检工具"，而非"导师替代品"。

- **导师不可替代的原因**：
  - 导师了解你的研究全貌、你的学术背景和研究过程
  - 导师能提供基于多年领域经验的深度指导
  - 导师能给予情感支持和针对性职业建议
  - 导师的学术人脉和资源对接无法通过 AI 实现

- **本工具的价值**：
  - 在导师时间有限的情况下，提供随时随地的答辩练习
  - 发现你自己未曾注意到的论文薄弱点
  - 帮助构建系统性答辩思路
  - 降低正式答辩前的焦虑感

**正确用法：** 将本工具作为导师指导的补充，而非替代。在导师指导之外，利用本工具进行高频次基础训练，带着 AI 模拟中发现的问题再去请教导师，效果更佳。

## 9. 模拟答辩的结果会影响我的真实成绩吗？

**答：** 不会。本工具是一个独立的练习辅助程序，不与任何学校、教育机构或学术委员会的数据系统关联。所有模拟答辩产生的评分、记录和反馈仅存在于你的本地设备或你主动使用的云服务中，不会影响你的真实学术成绩或毕业答辩评分。

---

## 10. 我可以用本工具模拟小组答辩或多人答辩吗？

**答：** 当前版本主要支持单人模拟答辩，即"一名学生 vs 多名 AI 评委"的模式。多人答辩（如多名学生同时答辩、小组质询等）场景暂不支持。未来版本可能会根据用户反馈考虑添加此功能。
