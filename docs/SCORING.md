# 答辩评分表与评估标准

## 一、评分总则

### 计分方式

本评分体系采用 **百分制** 与 **五级制** 并行的方式：

| 百分制分数 | 五级制等级 | 含义 |
|-----------|-----------|------|
| 90-100 分 | 优秀（A） | 远超基本要求，在多个维度表现突出 |
| 80-89 分  | 良好（B） | 达到要求且有一定亮点，仅有少量不足 |
| 70-79 分  | 中等（C） | 基本达到要求，但存在若干明显不足 |
| 60-69 分  | 及格（D） | 勉强达到最低要求，存在较多问题 |
| 0-59 分   | 不及格（F）| 未达到基本要求，存在严重缺陷 |

### 评分基本原则

1. **客观公正**：所有评分应基于论文实际内容和答辩表现，而非个人偏好
2. **综合评价**：最终得分由各维度加权汇总得出，任何单一维度不单独决定最终结果
3. **透明可追溯**：每项评分应有对应评价依据，避免主观臆断
4. **发展导向**：评分不仅用于评判，更应指出改进方向

## 二、评分维度与权重

| 维度编号 | 维度名称 | 权重 | 说明 |
|---------|---------|------|------|
| D1 | 选题意义 | 15% | 选题的理论价值、现实意义及创新性 |
| D2 | 理论框架 | 15% | 文献综述的全面性、理论基础的恰当性 |
| D3 | 研究设计 | 20% | 研究方法的科学性、合理性、可操作性 |
| D4 | 数据分析 | 20% | 数据收集与处理的质量、分析方法的恰当性 |
| D5 | 结论与贡献 | 15% | 研究结论的可靠性、理论/实践贡献 |
| D6 | 学术规范 | 10% | 论文结构、引用规范、语言表达 |
| D7 | 答辩表现 | 5% | 语言表达、问题应对、时间控制 |
| **合计** | | **100%** | |

## 三、各维度评分细则

### D1：选题意义（权重 15%）

| 等级 | 分数 | 标准描述 |
|------|------|----------|
| **优秀** | 13-15 | 选题具有重要的理论价值或现实意义；创新性强，填补了研究空白或提出了新视角；研究问题聚焦、明确，具有可研究性 |
| **良好** | 11-12 | 选题具有一定理论和现实意义；存在一定创新性；研究问题较为清晰，基本可操作 |
| **及格** | 9-10 | 选题有意义但较为常规；创新性不明显；研究问题基本明确但稍显宽泛 |
| **不及格** | 0-8 | 选题缺乏学术价值或现实意义；无创新性；研究问题模糊或过于宏大难以操作 |

### D2：理论框架（权重 15%）

| 等级 | 分数 | 标准描述 |
|------|------|----------|
| **优秀** | 13-15 | 文献综述全面系统，涵盖核心理论与最新进展；引用的理论框架与研究问题高度契合；对已有研究有批判性思考，清晰论证研究定位 |
| **良好** | 11-12 | 文献综述较为全面，涵盖主要理论与相关研究；理论框架基本合理；能指出已有研究的不足，但分析深度有限 |
| **及格** | 9-10 | 文献综述覆盖了基本文献但不够系统；理论框架选择较为随意；对已有文献的评述较为表面 |
| **不及格** | 0-8 | 文献综述明显不完整，遗漏关键文献；没有明确的理论框架或框架与研究问题不匹配；缺乏对文献的批判性分析 |

### D3：研究设计（权重 20%）

| 等级 | 分数 | 标准描述 |
|------|------|----------|
| **优秀** | 18-20 | 研究方法选择恰当，论证充分；研究设计严谨，变量/概念操作化清晰；样本/案例选择合理且论证充分；研究方法描述详尽，可复现 |
| **良好** | 15-17 | 研究方法基本合适；研究设计较为完整，存在少量瑕疵；样本/案例选择基本合理；研究方法的描述较为清晰 |
| **及格** | 12-14 | 研究方法选择基本合理但论证不足；研究设计存在若干漏洞；样本/案例选择带有明显局限性；方法描述不够充分 |
| **不及格** | 0-11 | 研究方法不适用于研究问题；研究设计存在严重缺陷；样本/案例选择有明显偏差；方法描述缺失或严重不足 |

### D4：数据分析（权重 20%）

| 等级 | 分数 | 标准描述 |
|------|------|----------|
| **优秀** | 18-20 | 数据收集过程规范，数据质量高；分析方法恰当且运用熟练；分析结果呈现清晰、图/表规范；对分析结果有深入解读，能发现数据中的深层含义 |
| **良好** | 15-17 | 数据收集基本规范；分析方法基本合适；结果呈现较为清晰；对结果进行了合理解读，但深度一般 |
| **及格** | 12-14 | 数据收集过程存在不足；分析方法选择有争议或运用不够规范；结果呈现不够清晰；结果解读较为表面 |
| **不及格** | 0-11 | 数据收集存在严重问题（如样本量过小、数据来源不可靠）；分析方法明显错误；结果呈现混乱；缺乏对结果的有效解读 |

### D5：结论与贡献（权重 15%）

| 等级 | 分数 | 标准描述 |
|------|------|----------|
| **优秀** | 13-15 | 结论基于分析结果，论证充分；回答了所有研究问题；明确阐述了理论和/或实践贡献；客观讨论了研究局限性及未来方向 |
| **良好** | 11-12 | 结论基本合理，与结果一致；对研究问题有回答但不够全面；指出了贡献但阐述不够具体；提及了局限性但分析较浅 |
| **及格** | 9-10 | 结论与结果基本一致但不够扎实；对研究问题的回答不完整；贡献陈述较为笼统；缺乏对局限性的讨论 |
| **不及格** | 0-8 | 结论与数据结果不一致或过度推论；没有回答研究问题；无法说明研究有何贡献；未讨论局限性 |

### D6：学术规范（权重 10%）

| 等级 | 分数 | 标准描述 |
|------|------|----------|
| **优秀** | 9-10 | 论文结构完整、逻辑清晰；引用规范，标注准确；语言流畅、表达精准；格式排版规范统一 |
| **良好** | 7-8 | 论文结构基本完整；引用基本规范但少量标注有误；语言表达较为流畅；排版基本规范 |
| **及格** | 6 | 论文结构有瑕疵（如缺少必要章节）；引用存在较多不规范之处；语言表达有含混之处；格式存在不统一 |
| **不及格** | 0-5 | 论文结构混乱；引用存在抄袭或大量不规范；语言表达差，影响理解；格式严重混乱 |

### D7：答辩表现（权重 5%）

| 等级 | 分数 | 标准描述 |
|------|------|----------|
| **优秀** | 5 | 陈述清晰有条理；回答问题准确、自信、有层次；时间控制得当；能够有效回应质疑 |
| **良好** | 4 | 陈述基本清晰；能回答大部分问题但深度不足；时间控制基本合理；面对质疑能基本回应 |
| **及格** | 3 | 陈述不够流畅；部分问题回答不到位；时间控制略显紧张；面对质疑应对不够充分 |
| **不及格** | 0-2 | 陈述混乱；无法回答核心问题；时间严重超时或过短；面对质疑无法有效回应 |

## 四、综合评定等级说明

| 综合等级 | 百分制范围 | 评定标准 |
|---------|-----------|---------|
| **优秀（A）** | 90-100 分 | 在所有维度上表现突出，论文质量达到或接近发表水平，答辩表现优秀。同意通过，建议推荐为优秀论文。 |
| **良好（B）** | 80-89 分 | 在大部分维度上表现良好，仅有少量不足。同意通过，但建议根据修改意见完善细节。 |
| **中等（C）** | 70-79 分 | 基本达到学术要求，但存在若干需改进的问题。有条件通过，需在规定时间内完成修改。 |
| **及格（D）** | 60-69 分 | 勉强达到最低学术标准，但存在较多明显问题。需进行重大修改后重新提交或参加二次答辩。 |
| **不及格（F）** | 0-59 分 | 存在严重缺陷，未能达到学位论文基本要求。不予通过，建议重新选题或大幅修改后在下一周期重新申请答辩。 |

### 一票否决情形

以下任一情形将直接判定为 **不及格（F）**，不适用加权评分：

1. 经查实存在学术不端行为（抄袭、数据造假、一稿多投等）
2. 研究违反学术伦理（如未获伦理审查批准即开展涉及人类/动物的实验）
3. 论文核心内容与提交的最终版存在实质性差异
4. 答辩过程中出现严重不当言行

### 评委综合意见（示例）

> 该生的论文选题具有一定的现实意义，研究方法设计合理，整体结构完整，符合学术规范。答辩过程中表达清晰，能够较好地回应评委提问。
>
> **主要建议：**
> 1. 进一步明确理论框架与研究方法之间的逻辑关联
> 2. 补充样本选择的论证依据和局限性说明
> 3. 深化实践贡献的阐述，将其与具体研究发现挂钩
> 4. 补充关于缺失数据处理的技术说明
>
> **评定结果：良好（B），同意通过答辩。**
